Desafio de Processo Seletivo

Bem-vindo ao sistema de gerenciamento de usuários e perfis! Este projeto faz parte de um desafio de processo seletivo e tem como objetivo atender os seguintes requisitos:

  • Criar um sistema que gerencie usuários e perfis.
  • Usuário possui um perfil; um perfil pode ter vários usuários.
  • O sistema deverá ter um administrador que crie os usuários e atribua ou modifique os perfis.
  • O perfil "usuário comum" apenas visualizará suas próprias informações, podendo editá-las, exceto o perfil.
  • Frontend desenvolvido em Angular 4+.
  • Backend utilizando Java 8+ com Spring Boot e banco de dados de sua escolha.

Este projeto demonstra habilidades em desenvolvimento full-stack, incluindo a implementação de segurança, gestão de dados e interface de usuário amigável.
